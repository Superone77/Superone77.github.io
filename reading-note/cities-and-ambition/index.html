城市与雄心 #

伟大的城市总能吸引胸怀大志的人。当你漫步其中，便能真切感受到这一点。城市会以无数种微妙的方式，向你传递同一条讯息：你还能做得更多，你应该再拼一把。

最令人意外的是，这些讯息竟如此千差万别。纽约向你传递的，首要的一句话就是：你该赚更多的钱。当然，这里也有其他的声音：你该更潮，你该更好看。但最清晰、最响亮的，永远是那句“你该变得更富有”。

我最喜欢波士顿（更准确地说是剑桥市）的一点，是它传递的讯息：你该更聪明。你真该把那些一直想读的书，都好好读一遍。

当你追问一座城市传递的核心讯息时，有时会得到意想不到的答案。硅谷的人们固然极度尊重才智，但硅谷向你传递的讯息却是：你该变得更有影响力。

这和纽约的讯息并不完全相同。当然，权力在纽约同样重要，但哪怕你只是继承了十亿美元，纽约也会对你刮目相看。可在硅谷，除了几个房产中介，没人会在意这种事。在硅谷，真正重要的，是你对这个世界产生了多大的影响。人们之所以关注拉里和谢尔盖，并非因为他们的财富，而是因为他们掌控着谷歌，而这家公司几乎影响着每一个人。

一座城市传递的讯息，到底有多重要？从实际经验来看，答案是：至关重要。你或许会想，只要你有足够坚定的意志去成就伟业，就能超越周遭的环境。你居住的地方，最多也就带来百分之几的影响。但如果你看看历史上的案例，就会发现它的影响远不止于此。绝大多数成就伟业的人，都聚集在少数几个地方——那些在特定时代，孕育着同类事业的地方。

从一件我此前写过的事里，就能看出城市的力量有多强大：米兰出生的达芬奇的故事。15世纪意大利所有你叫得出名字的画家，几乎都来自佛罗伦萨，尽管当时的米兰和佛罗伦萨规模相当。佛罗伦萨人并非天生天赋异禀，所以我们必须承认，当年一定有个和达芬奇天赋相当的人，出生在米兰。他后来怎么样了？

如果连拥有和达芬奇同等天赋的人，都无法对抗环境的力量，你觉得你能吗？

我做不到。我自认是个相当固执的人，但我绝不会试图对抗这种力量，我更愿意借势而为。所以，我花了很多心思思考，该住在哪里。

我一直以为，伯克利会是理想之地——它本质上就是个天气更好的剑桥。但几年前，当我真的在那里住了一段时间后，才发现事实并非如此。伯克利传递的讯息是：你该活得更好。伯克利的生活极具人文气息，这里或许是全美国最能让北欧人有归属感的地方。但这里，没有那种雄心壮志的蓬勃脉动。

现在回头看，一个如此宜居的地方，最终吸引来的，终究是那些把生活品质放在第一位的人，这本就不足为奇。事实证明，拥有好天气的剑桥，终究不是剑桥。你在剑桥遇到的人，并非偶然出现在那里。为了住在那里，你必须做出牺牲。这里物价高昂，环境算不上整洁，天气也常常很糟糕。所以，你在剑桥遇到的人，都是那些甘愿为了和最聪明的人为邻，而忍受高物价、脏乱环境和坏天气的人。

在我写下这篇文章时，剑桥似乎仍是世界的智力之都。我知道这个说法听起来很荒谬，但更荒谬的，是把这个称号安在其他任何地方。从满怀雄心的学子流向来看，美国的大学目前显然是全球顶尖的。那么，美国还有哪座城市，更有资格担起这个称号？纽约？那里确实有不少聪明人，但早已被数量多得多的西装革履的平庸之辈稀释了。湾区也有很多聪明人，但同样被稀释了；这里有两所顶尖大学，却相隔甚远。而按西海岸的标准来看，哈佛和MIT几乎是紧挨着的，它们周边还环绕着另外20多所高等院校。[1]

因此，剑桥给人的感觉，是一座以思想为核心产业的城市；而纽约的核心产业是金融，硅谷的则是创业。

当我们在这里谈论城市时，我们真正谈论的，是人的聚集。长久以来，城市是唯一能实现大规模人群聚集的地方，所以我们可以把这两个概念等同起来。但从我前面举的例子里，就能看出世事变迁有多快。纽约是经典的伟大城市，可剑桥只是一座城市的一部分，硅谷甚至连这个都算不上。（圣何塞有时会自称是硅谷之都，但事实并非如此。它不过是硅谷一端，一片178平方英里的区域而已。）

或许互联网会进一步改变这一切。或许有一天，你所属的最重要的社群，会是一个虚拟社群，而你实际住在哪里，将变得无足轻重。但我不会对此下注。现实世界的信息带宽极高，而城市传递讯息的方式，往往极其微妙。

每年春天回到剑桥，最令人心旷神怡的一件事，就是黄昏时分漫步街头，能瞥见屋内的景象。傍晚时分走在帕洛阿尔托，你看到的只有电视屏幕发出的蓝光。而在剑桥，你看到的是摆满了令人心生向往的书籍的书架。1960年的帕洛阿尔托，或许和如今的剑桥很像，但现在，你根本猜不到这里附近有一所大学。如今这里，不过是硅谷里一个富人聚居区罢了。[2]

一座城市对你的低语，大多是不经意间发生的——透过窗户看到的景象，无意间听到的对话。你不必刻意去寻找，它却无孔不入，你根本无法屏蔽。住在剑桥的一个“职业风险”，就是总会听到有人用疑问句的语调，说陈述句。但总的来说，比起纽约和硅谷的对话，我还是更愿意听剑桥的。

90年代末搬到硅谷的一个朋友跟我说，住在那里最糟糕的事，就是“偷听”到的对话质量太低。当时我还觉得，她是故意标新立异。当然，偷听别人说话可能会很有趣，但高质量的偷听内容，真的重要到能影响你对居住地的选择吗？现在我懂她的意思了。你无意间听到的对话，会告诉你，你身边都是什么样的人。

无论你的意志有多坚定，都很难不被周围的人影响。倒不是说你会完全按照城市对你的期待去做事，而是当身边没有一个人关心你所在乎的事时，你会变得气馁。

这种鼓励与打击之间的失衡，就像赚钱与亏钱之间的失衡一样。绝大多数人都会高估亏损的影响：他们为了避免损失1美元付出的努力，远比为了赚到1美元付出的多。同样，尽管有很多人意志坚定，不会仅仅因为身处某地、大家都那么做，就随波逐流；但几乎没有人能强大到，在身边所有人都不在乎自己所做之事时，还能一直坚持下去。

因为不同的野心，在某种程度上是互斥的，而他人的敬佩，是一场零和博弈。所以，每座城市往往只会聚焦于一种野心。剑桥之所以能成为智力之都，不仅是因为这里聚集了大量聪明人，更因为在这里，没有什么事比智力更受人们重视。在纽约和湾区，教授只是二等公民——除非他们分别创办了对冲基金，或是科技创业公司。

这也回答了互联网泡沫破裂后，纽约人一直好奇的一个问题：纽约能否成长为能与硅谷抗衡的创业中心？这件事大概率不会发生，其中一个原因就是，在纽约创业的人，会觉得自己是二等公民。[3] 在这里，早已另有一件事，赢得了人们更多的推崇。

长远来看，这对纽约而言可能不是什么好事。一项重要新技术的力量，最终总会转化为财富。所以，比起硅谷，纽约更看重金钱、而非影响力，本质上是对同一件事的追逐，只是慢了一步。[4] 而事实上，在这场它自己最擅长的游戏里，纽约正在输给硅谷：福布斯美国400富豪榜中，纽约与加州的上榜人数之比，已经从1982年榜单首次发布时的1.45（81:56），降到了2007年的0.83（73:88）。

并非所有城市都会传递讯息。只有那些成为某种野心聚集地的城市，才会发出自己的声音。而不亲身居住其中，你往往很难准确判断一座城市传递的讯息到底是什么。我能读懂纽约、剑桥和硅谷的讯息，是因为我在每一个地方都住了好几年。华盛顿和洛杉矶似乎也有自己的讯息，但我在这两个地方待的时间都不够长，无法给出确切的答案。

洛杉矶最核心的追求，似乎是名气。这里有一张当下最炙手可热的一线明星榜单，最受人推崇的事，就是跻身其中，或是和榜上的人成为朋友。除此之外，这里的讯息和纽约大同小异，只是或许更看重外貌吸引力。

而在华盛顿，传递的讯息似乎是：最重要的事，是你的人脉。你要成为圈内人。实际上，这和洛杉矶的逻辑几乎如出一辙：同样有一张一线榜单，你要么跻身其中，要么靠近榜上的人。唯一的区别，是这张榜单的筛选标准。而即便是这个标准，其实也没那么大的不同。

如今，旧金山传递的讯息，似乎和伯克利一样：你该活得更好。但如果有足够多的创业公司选择旧金山而非硅谷，这一切将会改变。互联网泡沫时期，这是失败的前兆——一种自我放纵的选择，就像买昂贵的办公家具一样。即便是现在，看到创业公司选在旧金山，我依然会心生疑虑。但如果有足够多优秀的创业公司这么做，它就不再是自我放纵的选择了，因为硅谷的重心，将会转移到那里。

我还没找到任何一个地方，能像剑桥这样，孕育着对智力成就的极致追求。英国的牛津和剑桥，给人的感觉就像伊萨卡或是汉诺威：那里有同样的讯息，但强度要弱得多。

巴黎曾是伟大的智力中心。如果你在1300年去到那里，它传递的讯息，或许和如今的剑桥别无二致。但去年我在那里小住了一段时间，发现当地居民的野心，早已与智力无关。如今的巴黎，传递的讯息是：做事要有格调。说实话，我很喜欢这一点。巴黎是我住过的城市里，唯一一个人们真正发自内心在乎艺术的地方。在美国，只有少数富人会买原创艺术品，即便是更有鉴赏力的人，也大多只会通过艺术家的名气来评判作品。但在巴黎，黄昏时分透过窗户望去，你会发现，那里的人们是真的在意画作本身的样子。从视觉层面来说，巴黎拥有我所知道的最高质量的“偷听”体验。[5]

我还从城市里听到过另一种讯息：在伦敦，你依然（勉强）能听到这样的声音：人该活得更有贵族风范。如果你刻意去听，在巴黎、纽约和波士顿，也能捕捉到这种声音。但这种讯息，在所有地方都已经非常微弱了。放在100年前，它会无比响亮，但如今，若不是我刻意调谐到这个频段，看看是否还有信号，我大概率根本不会注意到它的存在。

到目前为止，我从各个城市里捕捉到的完整讯息清单如下：财富、格调、潮流、外貌、名气、政治权力、经济影响力、智力、社会阶层，以及生活品质。

看到这份清单，我的第一反应，是些许不适。我一直认为，雄心壮志是件好事，但现在我才意识到，这只是因为我潜意识里，只把自己所在乎领域的野心，当成了真正的野心。当你把所有胸怀大志之人的追求都列出来时，这件事就没那么美好了。

仔细审视这份清单，我发现有几项内容，放在历史的维度里看，十分令人意外。比如，外貌吸引力，在100年前的清单里，大概率不会出现（尽管在2400年前，它或许曾在列）。这件事对女性而言一直很重要，但在20世纪末，它似乎对男性也开始变得重要起来。我不确定其中的原因——或许是女性权力的提升、演员作为偶像的影响力越来越大，以及如今绝大多数人都在办公室工作的现状，共同作用的结果：你没法穿工厂里不能穿的华丽衣服来炫耀，所以只能用自己的身体来彰显优势。

潮流，是另一个100年前不会出现在清单里的东西。或者说，它其实一直都在？潮流的本质，是洞悉事物的本质与风向。所以，它或许只是取代了社会阶层中，那种“深谙圈内规则”的成分。这也能解释，为什么潮流在伦敦格外受推崇：它不过是传统英国人热衷的、只有圈内人才能懂的隐秘规则的2.0版本罢了。

经济影响力，在100年前的清单里必然占有一席之地，但如今它的内涵已经发生了变化。过去，它指的是对庞大人力与物质资源的掌控。但如今，它越来越多地意味着引领技术发展方向的能力，而有些拥有这种能力的人，甚至并不富有——比如重要开源项目的负责人。过去的工业巨头，拥有满实验室的聪明人，为他们研发新技术。而新一代的掌控者，本身就是这些聪明人。

随着这股力量越来越受关注，另一项内容正在从清单上消失：社会阶层。我认为这两个变化是相关的。经济影响力、财富和社会阶层，不过是同一件事在不同生命周期的名字：经济影响力转化为财富，财富再转化为社会阶层。所以，人们推崇的焦点，不过是在向上游转移罢了。

难道所有想成就一番伟业的人，都必须住在伟大的城市里吗？不。所有伟大的城市，都会激发某种形式的雄心，但它们并非唯一能做到这一点的地方。对于某些类型的工作而言，你只需要一小群有才华的同事就够了。

城市提供的，是受众，以及同路人的汇聚渠道。这些东西，在数学、物理这类领域，并没有那么关键。在这些领域，除了同行，其他受众都无关紧要，而且能力的评判标准足够直接，招聘和招生委员会就能做出可靠的判断。在数学或物理这类领域，你只需要一个有合适同行的院系就够了。它可以在任何地方——比如新墨西哥州的洛斯阿拉莫斯。

而在艺术、写作、科技这类领域，大环境就变得至关重要了。在这些领域，最顶尖的从业者，并不会整齐地聚集在少数几个顶尖大学的院系和实验室里——部分原因是天赋更难评判，另一部分原因是，这些领域的工作能直接带来收入，人们不需要依靠教学或科研经费来养活自己。正是在这些更具不确定性的领域，身处伟大的城市，才能给你带来最大的帮助：你需要那种身边有人在乎你所做之事的鼓励，而既然你需要自己去寻找同路人，你就需要伟大城市所提供的，那个庞大得多的人才汇聚机制。

你不必一辈子都住在伟大的城市里，才能从中受益。职业生涯的早期和中期，似乎才是最关键的阶段。显然，你不必在伟大的城市里长大，在那里上大学，似乎也没那么重要。对大多数大学生来说，一个几千人的世界，就已经足够大了。更何况，在大学阶段，你还不必面对最艰难的工作——发现那些有待解决的新问题。

当你迈入下一个、也艰难得多的阶段时，身处一个能找到同路人与鼓励的地方，才会给你带来最大的帮助。而一旦你同时拥有了这两者，如果你愿意，就可以离开了。印象派画家的经历，就是最典型的例子：他们出生在法国各地（毕沙罗甚至出生在加勒比海），也终老在法国各地，但真正定义了他们的，是那些在巴黎一同度过的岁月。

除非你确切地知道自己想做什么，也知道这个领域的顶尖中心在哪里，否则，对你来说最好的选择，或许就是年轻时多去几个地方住一住。 不亲身居住其中，你永远无法知道一座城市会传递什么样的讯息，甚至无法知道它是否还在发出自己的声音。你的认知往往会出错：我25岁时曾在佛罗伦萨住过一段时间，以为这里会是艺术中心，结果发现，我晚来了450年。

即便一座城市依然是孕育雄心的鲜活中心，你也要亲耳听到它的讯息，才能确定它是否能与你产生共鸣。我刚搬到纽约的时候，一开始无比兴奋。那是个令人心潮澎湃的地方。所以，我花了很长时间才意识到，我和那里的人，根本不是一类人。我一直在纽约寻找属于它的“剑桥”，结果发现，它远在市区的最北端——坐飞机都要一个小时。

有些人16岁就知道自己一生要做什么，但对大多数心怀大志的年轻人来说，往往是先有了雄心，才找到值得自己倾注雄心的具体目标。他们知道自己想成就一番伟业，只是还没决定，是要成为摇滚明星，还是脑外科医生。这没什么不对。但这也意味着，如果你拥有这种最常见的雄心，你大概率只能通过不断尝试，来找到适合自己的居住地。你或许只有找到那座让你有归属感的城市，才能知道自己真正的雄心，到底是什么。

注释 #

我看完了这篇《Cities and Ambitions》，有一些让我感触很深的点，我想从文中的一些摘录出发来聊聊。

  1. 城市的重要性与虚拟社群 作者提到，也许将来有一天互联网会改变城市的重要性。也就是说，你所属的重要社群可能是一个虚拟社群，而你住在哪里变得无足轻重。 但作者认为这一天并不会轻易到来，因为现实世界的信息带宽非常高，城市传递信息的方式极其微妙，虚拟社群很难实现这一点。

  2. 城市传递信息的方式 我觉得这非常有道理。一个城市传递信息的方式，可能通过整个城市的结构、城市里存在的人，或者城市所构造的整体氛围。这都是一种潜移默化的信息传递。

  3. 关于对话质量的例子 作者举了一个例子：一位在 90 年代末从剑桥搬到硅谷的朋友跟他抱怨，在硅谷让他觉得非常糟糕的一件事，就是偷听到的对话质量。相比于剑桥要低得多。因为你无意间听到的对话，就告诉了你身边都是什么样的人，以及你选择居住在了什么样的地方。当你身边没有一个人关心你所在乎的事的时候，你会变得气馁，无论你的意志有多坚定。

  4. 每个城市往往只会聚焦在一种野心上。剑桥更多专注在智力上，纽约专注在财力上，湾区则专注在创新和影响力上比方说一个教授，他在剑桥可能会备受尊敬，但在纽约和湾区，他们只是二等公民，除非他们分别创办了对冲基金或者是科技创业公司。

  5. 保罗对年轻人的建议是：除非你确切地知道自己想要做什么，也知道这个领域的顶尖中心在哪里，否则对于年轻人来说，最好的选择就是多去几个地方住一住。因为如果你不亲身居住在其中，你永远无法知道一座城市会传递什么样的讯息。当你到达一个合适的城市时，你可以找到同路人和鼓励，他们会给你带来非常大的帮助。并且这些帮助在你离开之后，也会不断地陪伴着你。比方说，他举了一个印象派画家的例子：印象派画家们出生在法国各地，也终老在法国各地，但真正定义了他们的是那些他们在巴黎一同度过的岁月。

  6. 我个人的感悟是，自从我从家乡出发之后，我主要待过三个城市：一个是北京，一个是达姆施塔特，还有一个是深圳。我在北京和深圳都找到了同路人和鼓励。在这两个地方度过的时光让我感到很振奋，它们对我的影响会持续到我离开这两座城市之后很长的时间。但相对来说，在达姆施塔特，我不觉得那个地方给我留下了什么深刻的印记，那段经历主要只是磨练了我的坚强，自信和耐心，但城市本身并没有给我带来太大的影响。所以我觉得年轻人还是应该更多地待在大城市。大城市确实会有很多机会，能让你找到合适的同伴。大城市的开放与激励，能够让年轻人找到自己真正感兴趣的东西。
