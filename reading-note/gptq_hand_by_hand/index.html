第一步：实现均匀对称量化 #

1. 计算原范围

ori_range = x.abs().max()

2. 计算量化精度所能表示的范围

bits_range = 2**(bits - 1) - 1 为什么？以int4举例

计算规则：

  • int4可写作 b3 b2 b1 b0
  • 十进制值 = -8*b3 + 4*b2 + 2*b1 + 1*b0
  • 其中最高位 b3 是符号位，权重是 -8（不是 +8）

int4 全部取值表：

0100 -8*0 + 4*1 + 2*0 + 1*0 4

所以 int4 范围是：-8 ~ 7。
如果你愿意，我还可以再给你一版“用补码取反加一”来验证负数的推导过程。

3. 计算缩放因子

scale = ori_range / bits_range

4. 量化 + 反量化

x_q = (x / scale).round() x_q = x_q * scale return x_q

第二步：实现Hessian矩阵 #

什么是黑森矩阵，如何计算一个线性层输入的黑森矩阵？ 黑森矩阵（Hessian）是一个函数对参数的二阶偏导数组成的矩阵，用来描述“曲率”。 对标量损失 (L(\theta))，Hessian 定义为： [ H_{ij}=\frac{\partial^2 L}{\partial \theta_i\partial \theta_j} ]

对角线大：该方向曲率大，参数改一点损失就变很多 非对角线大：两个参数之间耦合强（一起变化会影响损失）

在GPTQ中，使用如下近似 H ~= X^T X / N + damping * I 含义是： X：线性层输入样本矩阵，形状通常是 ([N, d_{in}]) N：样本数 (\frac{X^\top X}{N})：输入协方差（或二阶矩）近似，反映不同输入维度的相关性 (\lambda I)（damping）：阻尼项，让矩阵更稳定、更可逆，防止数值问题 最后通常还会算 (H^{-1})（H_inv），用于后续量化误差补偿 所以代码如下 def build_hessian(calib_x: torch.Tensor, damping: float = 1e-4): “”" Approximate Hessian for linear layer inputs: H ~= X^T X / N + damping * I Returns H and H_inv. “”" x_t = calib_x.T N = x_t.shape[0] I = torch.eye(x_t.shape[0]) H = calib_x @ x_t / N + damping * I H_inv = torch.linalg.inv(H) return H,H_inv

第三步：如何为一行权重做GPTQ逐列更新 #

把线性层权重矩阵中的“一行”做 GPTQ 量化，而不是像 RTN 一样直接一把 round 完。 GPTQ 的核心就在这里：按列（按元素）依次量化 + 误差补偿到后续列。

大致流程是： 复制当前行权重作为可更新向量 w_work 从第 j 个元素开始，取当前值 w_work[j] 对这个标量做 4-bit 量化，得到 q_j 计算量化误差 e_j = w_work[j] - q_j 用 h_inv 把这个误差“传播/补偿”到后面的元素（j+1…），让后续量化考虑前面误差 最终得到整行的量化结果 q_row

def gptq_quantize_row(w_row: torch.Tensor, h_inv: torch.Tensor, bits: int = 4): “”" Quantize one output row of W with GPTQ-style column-wise updates. “”" w_work = w_row.clone() for j in range(w_work.shape[0]): tmp = w_work[j] tmp_q = quantize_tensor_symmetric(tmp, bits) e_j = tmp - tmp_q w_work[j+1:] = w_work[j+1:] - e_j * h_inv[j+1:,j] w_work[j] = tmp_q return w_work

第四步：如何对整个线性层做GPTQ更新 #

用 calib_x 构建 Hessian 近似和逆矩阵：H, H_inv = build_hessian(calib_x, damping) 深拷贝原始 linear，避免改坏原模型 遍历每个输出通道（即 weight 的每一行 w_row） 对每一行调用 gptq_quantize_row(w_row, h_inv=H_inv, bits=bits) 得到量化行 把所有量化行拼回新的 weight bias 原样保留不变（题目要求） 返回这个量化后的 q_layer（由于Parameter没有copy()所以要用copy_()原地拷贝。

def gptq_quantize_linear( linear: nn.Linear, calib_x: torch.Tensor, bits: int = 4, damping: float = 1e-4, ): “”" Return a new quantized Linear layer using GPTQ. “”" # build hessian based on calibration data H, H_inv = build_hessian(calib_x, damping) q_layer = copy.deepcopy(linear) with torch.no_grad(): q_w = [] for r in range(linear.weight.shape[0]): q_r = gptq_quantize_row(linear.weight[r], h_inv=H_inv, bits=bits) q_w.append(q_r) q_layer.weight.copy_(torch.stack(q_w, dim=0)) return q_layer
