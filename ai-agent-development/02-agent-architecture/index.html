第二章：Agent 架构设计 #

理解 AI Agent 的核心组件、架构类型和通信机制

2.1 Agent 核心组件 #

AI Agent 由多个核心组件组成，每个组件负责特定的功能，相互协作完成 Agent 的任务。这些核心组件包括感知模块、规划模块、行动模块、记忆模块和反思模块。

2.1.1 感知模块（Perception） #

感知的作用

感知模块是 Agent 与环境的接口，负责从环境中获取信息并转换为 Agent 可以理解的内部状态表示。感知模块是 Agent 的"感官"，类似于人类的视觉、听觉、触觉等感知能力。

感知输入类型

感知模块可以处理多种类型的输入数据：

输入类型 数据示例 典型模型/技术
文本输入 用户消息、文档内容、源代码 LLM、BERT、NLP 模型
图像输入 摄像头图像、截图、UI 界面 CNN、ViT、YOLO
语音输入 用户语音指令、会议录音 Whisper、DeepSpeech、ASR
传感器数据 温度、湿度、GPS、加速度计 信号处理、机器学习

感知处理流程

感知模块的处理流程可以分为四个步骤：

数据采集 → 预处理 → 特征提取 → 状态表示
  1. 数据采集：从各种输入源采集原始数据
  2. 预处理：清洗、标准化、归一化数据
  3. 特征提取：提取关键特征（文本向量化、图像特征提取）
  4. 状态表示：将特征转换为 Agent 内部状态表示

感知挑战

  1. 噪声：输入数据包含噪声，影响感知准确性

    • 解决方案：使用数据清洗和滤波技术

  2. 不确定性：环境状态不完全可知，存在不确定性

    • 解决方案：使用概率模型和贝叶斯推理

  3. 多模态融合：如何融合不同类型的数据（文本、图像、语音）

    • 解决方案：使用多模态模型（如 CLIP、Flamingo）

  4. 实时性：感知需要实时响应，延迟要低

    • 解决方案：使用轻量级模型和边缘计算

from typing import Dict, Any
import openai
from PIL import Image

class PerceptionModule:
    def __init__(self, model="gpt-4"):
        self.client = openai.OpenAI(api_key="your-api-key")
        self.model = model

    def perceive_text(self, text: str) -> Dict[str, Any]:
        """感知文本输入"""
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": text}],
            temperature=0.0
        )
        return {
            "type": "text",
            "raw": text,
            "embedding": self._get_embedding(text),
            "understanding": response.choices[0].message.content
        }

    def perceive_image(self, image_path: str) -> Dict[str, Any]:
        """感知图像输入"""
        image = Image.open(image_path)
        # 使用视觉模型（如 GPT-4 Vision）理解图像
        response = self.client.chat.completions.create(
            model="gpt-4-vision-preview",
            messages=[
                {"role": "user", "content": "描述这张图片"},
                {"role": "user", "content": image}
            ],
            temperature=0.0
        )
        return {
            "type": "image",
            "raw": image_path,
            "description": response.choices[0].message.content
        }

    def perceive_sensor(self, sensor_data: Dict[str, float]) -> Dict[str, Any]:
        """感知传感器数据"""
        # 使用信号处理和机器学习处理传感器数据
        processed_data = self._preprocess_sensor(sensor_data)
        features = self._extract_features(processed_data)
        return {
            "type": "sensor",
            "raw": sensor_data,
            "processed": processed_data,
            "features": features
        }

    def _get_embedding(self, text: str) -> list:
        """获取文本向量"""
        response = self.client.embeddings.create(
            model="text-embedding-3-small",
            input=text
        )
        return response.data[0].embedding

    def _preprocess_sensor(self, data: Dict[str, float]) -> Dict[str, float]:
        """预处理传感器数据"""
        # 数据清洗、标准化、归一化
        return {k: (v - min(data.values())) / (max(data.values()) - min(data.values()))
                for k, v in data.items()}

    def _extract_features(self, data: Dict[str, float]) -> list:
        """提取传感器特征"""
        # 特征提取（统计特征、频域特征等）
        return list(data.values())

2.1.2 规划模块（Planning） #

规划的作用

规划模块负责制定行动计划，将 Agent 的目标分解为可执行的行动序列。规划模块是 Agent 的"大脑"，负责决策和策略制定。

规划类型

规划类型 核心问题 典型算法 应用场景
任务规划 如何将复杂任务分解为子任务 HTN、Task Graph、PDDL 任务调度、项目计划
路径规划 如何在状态空间中找到最优路径 A*、Dijkstra、RRT 机器人导航、游戏 AI
资源规划 如何合理分配和调度有限资源 线性规划、整数规划 资源调度、任务分配
时间规划 如何考虑时间约束进行规划 PDDL、调度算法 日程安排、生产计划

1. 经典规划算法

import heapq
from typing import List, Dict, Tuple, Optional

class AStarPlanner:
    """A* 规划算法"""

    def __init__(self, start: str, goal: str, graph: Dict[str, Dict[str, int]]):
        self.start = start
        self.goal = goal
        self.graph = graph

    def heuristic(self, node: str) -> int:
        """启发式函数（直线距离）"""
        # 这里使用简单的启发式，实际应用中需要更精确的估计
        return 0

    def search(self) -> Optional[List[str]]:
        """A* 搜索"""
        open_set = []
        heapq.heappush(open_set, (0, self.start))
        came_from = {}
        g_score = {self.start: 0}
        f_score = {self.start: self.heuristic(self.start)}

        while open_set:
            current_f, current = heapq.heappop(open_set)

            if current == self.goal:
                return self._reconstruct_path(came_from, current)

            for neighbor, cost in self.graph[current].items():
                tentative_g = g_score[current] + cost
                if neighbor not in g_score or tentative_g < g_score[neighbor]:
                    came_from[neighbor] = current
                    g_score[neighbor] = tentative_g
                    f = tentative_g + self.heuristic(neighbor)
                    f_score[neighbor] = f
                    heapq.heappush(open_set, (f, neighbor))

        return None

    def _reconstruct_path(self, came_from: Dict[str, str], current: str) -> List[str]:
        """重构路径"""
        path = [current]
        while current in came_from:
            current = came_from[current]
            path.append(current)
        return path[::-1]

# 使用示例

    "A": {"B": 5, "C": 1},
    "B": {"D": 2},
    "C": {"B": 3, "D": 8},
    "D": {}
}

class LLMPlanner:
    """基于大语言模型的规划器"""

    def __init__(self, model="gpt-4"):
        self.client = openai.OpenAI(api_key="your-api-key")
        self.model = model

    def plan(self, objective: str, context: str = "") -> Dict[str, Any]:
        """使用 CoT 进行规划"""
        prompt = f"""
        目标：{objective}

        上下文：{context}

        请使用思维链（Chain of Thought）进行规划：

        步骤 1：分析目标，理解需求
        步骤 2：将目标分解为子任务
        步骤 3：确定子任务的执行顺序和依赖关系
        步骤 4：为每个子任务选择合适的工具和参数
        步骤 5：输出完整的行动计划

        输出格式：
        {
            "analysis": "目标分析",
            "subtasks": ["子任务1", "子任务2", ...],
            "dependencies": {"子任务2": ["子任务1"], ...},
            "tools": {"子任务1": "工具1", ...},
            "plan": ["步骤1", "步骤2", ...]
        }
        """
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        return response.choices[0].message.content

    def replan(self, objective: str, current_state: str, error: str) -> Dict[str, Any]:
        """根据当前状态和错误重新规划"""
        prompt = f"""
        目标：{objective}
        当前状态：{current_state}
        错误：{error}

        请根据当前状态和错误重新规划：
        1. 分析错误原因
        2. 调整计划
        3. 输出新的行动计划
        """
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        return response.choices[0].message.content

2.1.3 行动模块（Action） #

行动的作用

行动模块负责执行具体操作，将 Agent 的决策转换为对环境的影响。行动模块是 Agent 的"手脚"，负责执行 Agent 的决策。

行动类型

行动类型 示例 技术实现
工具调用 调用 API、执行函数 HTTP、gRPC、函数调用
消息发送 发送消息给用户或其他 Agent 消息队列、WebSocket
状态更新 更新数据库、文件系统 SQL、NoSQL、文件操作

行动执行流程

行动选择 → 参数设置 → 执行 → 结果获取 → 错误处理
  1. 行动选择：从规划模块选择要执行的行动
  2. 参数设置：根据规划结果设置行动参数
  3. 执行：执行行动（调用工具、发送消息、更新状态）
  4. 结果获取：获取行动执行结果
  5. 错误处理：处理执行失败、重试、降级

代码示例（行动模块）

import requests
import time
from typing import Dict, Any, Optional

class ActionModule:
    def __init__(self):
        self.action_history = []

    def execute_action(self, action: Dict[str, Any]) -> Dict[str, Any]:
        """执行行动"""
        action_type = action.get("type")

        try:
            if action_type == "api_call":
                result = self._execute_api_call(action)
            elif action_type == "message_send":
                result = self._execute_message_send(action)
            elif action_type == "state_update":
                result = self._execute_state_update(action)
            else:
                result = {"success": False, "error": f"Unknown action type: {action_type}"}

            # 记录行动历史
            self.action_history.append({
                "action": action,
                "result": result,
                "timestamp": time.time()
            })

            return result

        except Exception as e:
            error_result = {
                "success": False,
                "error": str(e),
                "retry_possible": self._is_retry_possible(e)
            }
            self.action_history.append({
                "action": action,
                "result": error_result,
                "timestamp": time.time()
            })
            return error_result

    def _execute_api_call(self, action: Dict[str, Any]) -> Dict[str, Any]:
        """执行 API 调用"""
        url = action.get("url")
        method = action.get("method", "GET")
        headers = action.get("headers", {})
        data = action.get("data")
        timeout = action.get("timeout", 10)

        try:
            response = requests.request(
                method=method,
                url=url,
                headers=headers,
                json=data,
                timeout=timeout
            )
            return {
                "success": True,
                "status_code": response.status_code,
                "data": response.json() if response.headers.get('content-type', '').startswith('application/json') else response.text
            }
        except requests.exceptions.Timeout:
            return {
                "success": False,
                "error": "Request timeout",
                "retry_possible": True
            }
        except requests.exceptions.ConnectionError:
            return {
                "success": False,
                "error": "Connection error",
                "retry_possible": True
            }

    def _execute_message_send(self, action: Dict[str, Any]) -> Dict[str, Any]:
        """发送消息"""
        recipient = action.get("recipient")
        message = action.get("message")
        channel = action.get("channel")

        # 这里实现实际的消息发送逻辑
        # 例如：发送到消息队列、WebSocket、API 等
        print(f"Sending message to {recipient}: {message}")

        return {
            "success": True,
            "message_id": f"msg_{int(time.time())}",
            "recipient": recipient,
            "channel": channel
        }

    def _execute_state_update(self, action: Dict[str, Any]) -> Dict[str, Any]:
        """更新状态"""
        state_type = action.get("state_type")  # "database", "file", "memory"
        data = action.get("data")

        # 这里实现实际的状态更新逻辑
        # 例如：更新数据库、写入文件、更新内存等
        print(f"Updating {state_type} state: {data}")

        return {
            "success": True,
            "state_type": state_type,
            "updated_at": time.time()
        }

    def _is_retry_possible(self, error: Exception) -> bool:
        """判断是否可以重试"""
        # 网络错误、超时等可以重试
        return isinstance(error, (requests.exceptions.Timeout, requests.exceptions.ConnectionError))

    def get_action_history(self) -> list:
        """获取行动历史"""
        return self.action_history

# 使用示例

# 执行 API 调用

    "type": "api_call",
    "url": "https://api.example.com/data",
    "method": "GET"
})
print("API 调用结果：", api_result)

message_result = action_module.execute_action({

2.2 单 Agent 架构 #

单 Agent 架构是指单个 Agent 完成任务的架构设计，根据 Agent 的决策方式和规划能力，可以分为反应式、审慎式、混合式和学习型架构。

2.2.1 反应式架构（Reactive） #

核心原理

基于规则的 Agent，感知环境后直接根据预定义的规则执行行动，无需显式规划。

架构组件

感知 → 规则匹配 → 行动执行

代码示例（反应式架构）

class ReactiveAgent:
    def __init__(self):
        # 定义规则库
        self.rules = [
            {"if": "weather", "then": "query_weather"},
            {"if": "book_flight", "then": "book_flight"},
            {"if": "time", "then": "get_time"},
            {"if": "unknown", "then": "ask_human"}
        ]
        self.action_module = ActionModule()

    def perceive(self, user_input: str) -> str:
        """感知用户输入"""
        # 简单的关键词匹配
        for rule in self.rules:
            if rule["if"] in user_input:
                return rule["then"]
        return "ask_human"

    def decide_action(self, perception: str) -> Dict[str, Any]:
        """决定行动"""
        action_type = perception

        if action_type == "query_weather":
            return {"type": "api_call", "url": "https://api.weather.com/beijing"}
        elif action_type == "book_flight":
            return {"type": "api_call", "url": "https://api.flight.com/book"}
        elif action_type == "get_time":
            return {"type": "message_send", "message": f"Current time: {time.time()}"}
        else:
            return {"type": "message_send", "message": "Sorry, I don't understand."}

    def execute(self, user_input: str):
        """执行完整流程"""
        # 感知
        perception = self.perceive(user_input)
        print(f"Perception: {perception}")

        # 决定行动
        action = self.decide_action(perception)
        print(f"Action: {action}")

        # 执行行动
        result = self.action_module.execute_action(action)
        print(f"Result: {result}")

2.2.2 审慎式架构（Deliberative） #

核心原理

基于规划的 Agent，感知环境后进行显式规划，然后执行规划的行动。

架构组件

感知 → 规划 → 行动执行

代码示例（审慎式架构）

class DeliberativeAgent:
    def __init__(self):
        self.planner = LLMPlanner()
        self.action_module = ActionModule()

    def perceive(self, user_input: str) -> Dict[str, Any]:
        """感知用户输入"""
        return {
            "type": "text",
            "content": user_input
        }

    def plan(self, perception: Dict[str, Any]) -> List[Dict[str, Any]]:
        """制定行动计划"""
        objective = perception["content"]
        plan = self.planner.plan(objective)
        return plan

    def execute_plan(self, plan: List[Dict[str, Any]]):
        """执行计划"""
        for action in plan:
            print(f"Executing action: {action}")
            result = self.action_module.execute_action(action)
            print(f"Result: {result}")

            # 如果行动失败，重新规划
            if not result.get("success"):
                print("Action failed, replanning...")
                new_plan = self.planner.replan(
                    objective=plan.get("objective"),
                    current_state=plan.get("current_state"),
                    error=result.get("error")
                )
                return self.execute_plan(new_plan)

    def run(self, user_input: str):
        """运行完整流程"""
        # 感知
        perception = self.perceive(user_input)
        print(f"Perception: {perception}")

        # 规划
        plan = self.plan(perception)
        print(f"Plan: {plan}")

        # 执行计划
        self.execute_plan(plan)

2.2.4 学习型架构（Learning） #

核心原理

基于学习的 Agent，通过学习从经验中获得策略，持续改进性能。

架构组件

感知 → 学习（RL/IL）→ 策略 → 行动执行

代码示例（学习型架构）

import numpy as np

class LearningAgent:
    def __init__(self, state_size: int, action_size: int):
        self.state_size = state_size
        self.action_size = action_size
        self.q_table = np.zeros((state_size, action_size))
        self.learning_rate = 0.1
        self.discount = 0.95
        self.epsilon = 0.1

    def choose_action(self, state: int) -> int:
        """选择行动（ε-greedy 策略）"""
        if np.random.random() < self.epsilon:
            # 探索：随机选择
            return np.random.randint(self.action_size)
        else:
            # 利用：选择最优行动
            return np.argmax(self.q_table[state])

    def learn(self, state: int, action: int, reward: float, next_state: int):
        """学习（Q-Learning 更新）"""
        # Q-Learning 更新公式
        best_next_action = np.argmax(self.q_table[next_state])
        td_target = reward + self.discount * self.q_table[next_state][best_next_action]
        td_error = td_target - self.q_table[state][action]
        self.q_table[state][action] += self.learning_rate * td_error

    def run_episode(self, env, max_steps: int = 1000):
        """运行一个回合"""
        state = env.reset()
        total_reward = 0

        for step in range(max_steps):
            # 选择行动
            action = self.choose_action(state)

            # 执行行动
            next_state, reward, done, _ = env.step(action)

            # 学习
            self.learn(state, action, reward, next_state)

            # 更新状态
            state = next_state
            total_reward += reward

            if done:
                break

        return total_reward

    def train(self, env, n_episodes: int = 1000):
        """训练 Agent"""
        rewards = []

        for episode in range(n_episodes):
            reward = self.run_episode(env)
            rewards.append(reward)

            if (episode + 1) % 100 == 0:
                avg_reward = np.mean(rewards[-100:])
                print(f"Episode {episode + 1}, Average Reward: {avg_reward:.2f}")

        return rewards

# 使用示例（简单的环境）
class SimpleEnv:
    def __init__(self):
        self.state = 0
        self.n_states = 10
        self.n_actions = 2  # 0: 左, 1: 右

    def reset(self):
        self.state = 0
        return self.state

    def step(self, action):
        if action == 0:  # 左
            self.state = max(0, self.state - 1)
        else:  # 右
            self.state = min(self.n_states - 1, self.state + 1)

        reward = 1 if self.state == self.n_states - 1 else -0.1
        done = self.state == self.n_states - 1
        return self.state, reward, done, {}

# 创建并训练 Agent

        """注册子 Agent"""
        self.sub_agents[name] = agent

    def decompose_task(self, objective: str) -> List[Dict[str, Any]]:
        """分解任务"""
        plan = self.planner.plan(objective)
        return plan

    def assign_tasks(self, tasks: List[Dict[str, Any]]) -> Dict[str, Any]:
        """分配任务给子 Agent"""
        results = {}

        for task in tasks:
            agent_name = task.get("agent")
            agent = self.sub_agents.get(agent_name)

            if agent:
                print(f"Assigning task to {agent_name}: {task}")
                result = agent.execute(task)
                results[task["id"]] = result
            else:
                print(f"Agent not found: {agent_name}")
                results[task["id"]] = {"success": False, "error": "Agent not found"}

        return results

    def integrate_results(self, results: Dict[str, Any]) -> Dict[str, Any]:
        """整合子 Agent 的结果"""
        # 这里实现结果整合逻辑
        integrated_result = {
            "success": True,
            "subtask_results": results,
            "final_output": self._generate_final_output(results)
        }
        return integrated_result

    def _generate_final_output(self, results: Dict[str, Any]) -> str:
        """生成最终输出"""
        # 使用 LLM 整合结果
        results_summary = "\n".join([
            f"Subtask {k}: {v}" for k, v in results.items()
        ])

        prompt = f"""
        以下是各子任务的执行结果：
        {results_summary}

        请整合这些结果，生成最终的输出。
        """
        response = self.planner.client.chat.completions.create(
            model="gpt-4",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.3
        )
        return response.choices[0].message.content

    def run(self, objective: str) -> Dict[str, Any]:
        """运行完整流程"""
        print(f"Master Agent: Objective - {objective}")

        # 分解任务
        tasks = self.decompose_task(objective)
        print(f"Master Agent: Decomposed into {len(tasks)} subtasks")

        # 分配任务
        results = self.assign_tasks(tasks)
        print(f"Master Agent: All subtasks completed")

        # 整合结果
        final_result = self.integrate_results(results)
        print(f"Master Agent: Final result - {final_result}")

        return final_result

import concurrent.futures
from typing import List, Dict, Any

class ParallelCoordinator:
    def __init__(self):
        self.workers = {}
        self.executor = concurrent.futures.ThreadPoolExecutor(max_workers=4)

    def register_worker(self, name: str, worker):
        """注册 Worker"""
        self.workers[name] = worker

    def distribute_tasks(self, tasks: List[Dict[str, Any]]) -> Dict[str, Any]:
        """分发任务"""
        task_assignments = {}

        for i, task in enumerate(tasks):
            # 轮询分配任务
            worker_name = list(self.workers.keys())[i % len(self.workers)]
            task_assignments[i] = {
                "task": task,
                "worker": worker_name
            }

        return task_assignments

    def execute_parallel(self, task_assignments: Dict[str, Any]) -> Dict[str, Any]:
        """并行执行任务"""
        futures = {}
        results = {}

        # 提交任务到线程池
        for task_id, assignment in task_assignments.items():
            worker = self.workers[assignment["worker"]]
            future = self.executor.submit(worker.execute, assignment["task"])
            futures[task_id] = future

        # 等待所有任务完成
        for task_id, future in futures.items():
            try:
                result = future.result(timeout=60)
                results[task_id] = result
                print(f"Task {task_id} completed: {result}")
            except Exception as e:
                results[task_id] = {"success": False, "error": str(e)}
                print(f"Task {task_id} failed: {e}")

        return results

    def aggregate_results(self, results: Dict[str, Any]) -> Dict[str, Any]:
        """汇总结果"""
        # 这里实现结果汇总逻辑
        aggregated = {
            "total_tasks": len(results),
            "successful_tasks": sum(1 for r in results.values() if r.get("success")),
            "failed_tasks": sum(1 for r in results.values() if not r.get("success")),
            "results": results
        }
        return aggregated

    def run(self, tasks: List[Dict[str, Any]]) -> Dict[str, Any]:
        """运行完整流程"""
        print(f"Parallel Coordinator: Distributing {len(tasks)} tasks")

        # 分发任务
        task_assignments = self.distribute_tasks(tasks)
        print(f"Parallel Coordinator: Tasks distributed")

        # 并行执行
        results = self.execute_parallel(task_assignments)
        print(f"Parallel Coordinator: All tasks completed")

        # 汇总结果
        aggregated = self.aggregate_results(results)
        print(f"Parallel Coordinator: Results aggregated - {aggregated}")

        return aggregated

        elif team == "blue":
            self.blue_agent = agent

    def battle(self, rounds: int = 10) -> Dict[str, Any]:
        """执行对抗"""
        print(f"Starting adversarial battle: {rounds} rounds")

        for round in range(1, rounds + 1):
            print(f"\n--- Round {round} ---")

            # 红队行动
            red_action = self.red_agent.decide_action(self._get_state("red"))
            red_result = self._execute_action("red", red_action)
            print(f"Red Agent: {red_action} -> {red_result}")

            # 蓝队行动
            blue_action = self.blue_agent.decide_action(self._get_state("blue"))
            blue_result = self._execute_action("blue", blue_action)
            print(f"Blue Agent: {blue_action} -> {blue_result}")

            # 评分
            self._score_round(red_result, blue_result)

        print(f"\n--- Final Score ---")
        print(f"Red: {self.score['red']}, Blue: {self.score['blue']}")

        winner = "Red" if self.score['red'] > self.score['blue'] else "Blue"
        if self.score['red'] == self.score['blue']:
            winner = "Draw"

        return {
            "winner": winner,
            "score": self.score,
            "rounds": rounds
        }

        # 这里实现状态获取逻辑

        """执行行动"""
        # 这里实现行动执行逻辑

        self.nodes = {}
        self.global_model = None

    def register_node(self, node: FederatedNode):
        """注册节点"""
        self.nodes[node.node_id] = node

    def federated_learning(self, rounds: int = 3) -> Dict[str, Any]:
        """执行联邦学习"""
        print(f"Federated Coordinator: Starting {rounds} rounds of federated learning")

        for round in range(1, rounds + 1):
            print(f"\n--- Round {round} ---")

            # 本地训练
            local_models = []
            for node in self.nodes.values():
                local_model = node.train_local_model()
                local_models.append(local_model)

            # 聚合模型
            global_model = self._aggregate_models(local_models)
            print(f"Round {round}: Global model aggregated")

            # 分发全局模型
            self._distribute_global_model(global_model)

        return {
            "rounds": rounds,
            "final_global_model": self.global_model
        }

        """聚合本地模型"""
        # 这里实现模型聚合逻辑
        # 使用联邦平均（Federated Averaging）
        weights = [model["model_params"]["weights"] for model in local_models]
        biases = [model["model_params"]["bias"] for model in local_models]
        accuracies = [model["model_params"]["accuracy"] for model in local_models]

        # 聚合权重
        avg_weights = np.mean(weights, axis=0)
        avg_biases = np.mean(biases, axis=0)
        avg_accuracy = np.mean(accuracies)

        global_model = {
            "weights": avg_weights,
            "bias": avg_biases,
            "accuracy": avg_accuracy
        }

        self.global_model = global_model

        return global_model

    def _distribute_global_model(self, global_model: Dict[str, Any]):
        """分发全局模型"""
        # 这里实现全局模型分发逻辑
        for node in self.nodes.values():
            node.local_model = global_model
            print(f"Global model distributed to node {node.node_id}")

import asyncio
from typing import Dict, Any, Callable

class MessageBus:
    def __init__(self):
        self.subscribers = {}
        self.message_queue = asyncio.Queue()

    async def subscribe(self, topic: str, callback: Callable):
        """订阅主题"""
        if topic not in self.subscribers:
            self.subscribers[topic] = []
        self.subscribers[topic].append(callback)
        print(f"Subscribed to topic: {topic}")

    async def publish(self, topic: str, message: Dict[str, Any]):
        """发布消息"""
        print(f"Publishing to topic {topic}: {message}")
        await self.message_queue.put({"topic": topic, "message": message})

    async def start(self):
        """启动消息总线"""
        while True:
            item = await self.message_queue.get()
            topic = item["topic"]
            message = item["message"]

            if topic in self.subscribers:
                for callback in self.subscribers[topic]:
                    await callback(message)

class Agent:
    def __init__(self, name: str, message_bus: MessageBus):
        self.name = name
        self.message_bus = message_bus

    async def send_message(self, recipient: str, message: str):
        """发送消息"""
        await self.message_bus.publish(
            topic=f"agent.{recipient}",
            message={"sender": self.name, "content": message}
        )

    async def receive_message(self, message: Dict[str, Any]):
        """接收消息"""
        sender = message["sender"]
        content = message["content"]
        print(f"{self.name} received from {sender}: {content}")

    async def start(self):
        """启动 Agent"""
        await self.message_bus.subscribe(f"agent.{self.name}", self.receive_message)

# 使用示例
async def main():
    message_bus = MessageBus()

    # 创建 Agent
    agent1 = Agent("Agent1", message_bus)
    agent2 = Agent("Agent2", message_bus)
    agent3 = Agent("Agent3", message_bus)

    # 启动 Agent
    await asyncio.gather(
        agent1.start(),
        agent2.start(),
        agent3.start()
    )

    # 启动消息总线
    bus_task = asyncio.create_task(message_bus.start())

    # 发送消息

    await agent1.send_message("Agent3", "Hello Agent3!")

    # 等待消息处理

    # 停止消息总线
    bus_task.cancel()

import openai
from typing import List, Dict, Any

class HumanInterface:
    def __init__(self, model="gpt-4"):
        self.client = openai.OpenAI(api_key="your-api-key")
        self.model = model
        self.conversation_history = []

    def send_message(self, message: str, conversation_id: str = None) -> Dict[str, Any]:
        """发送消息"""
        # 添加到对话历史
        self.conversation_history.append({"role": "user", "content": message})

        # 调用 LLM
        response = self.client.chat.completions.create(
            model=self.model,
            messages=self.conversation_history,
            temperature=0.7
        )

        assistant_message = response.choices[0].message.content

        # 添加到对话历史
        self.conversation_history.append({"role": "assistant", "content": assistant_message})

        return {
            "message": assistant_message,
            "conversation_id": conversation_id
        }

    def get_visualization(self, data: Dict[str, Any], chart_type: str = "bar") -> Dict[str, Any]:
        """生成可视化"""
        # 这里实现可视化生成逻辑
        # 返回可视化配置或图表 URL
        return {
            "chart_type": chart_type,
            "data": data,
            "url": f"https://example.com/charts/{chart_type}"
        }

# 使用示例
interface = HumanInterface()

# 自然语言对话
response = interface.send_message("帮我分析一下销售数据")
print("Agent 回复：", response)

# 可视化界面
chart = interface.get_visualization({"Q1": 100, "Q2": 150, "Q3": 200, "Q4": 250}, "bar")
print("可视化：", chart)

小结 #

本章介绍了 AI Agent 的架构设计，包括核心组件、单 Agent 架构、多 Agent 架构和通信机制。

Agent 核心组件

  • 感知模块：从环境中获取信息并转换为内部状态表示
  • 规划模块：制定行动计划，分解任务
  • 行动模块：执行具体操作，将决策转换为环境影响
  • 记忆模块：存储和检索历史信息
  • 反思模块：从经验中学习，提供改进建议

单 Agent 架构

  • 反应式架构：基于规则，响应快，但缺乏规划
  • 审慎式架构：基于规划，适应性强，但响应慢
  • 混合式架构：结合反应式和审慎式，兼顾响应和规划
  • 学习型架构：基于学习，自适应，但需要大量数据

多 Agent 架构

  • 层次化架构：主 Agent 分解任务，子 Agent 执行任务
  • 并行协作架构：多个 Agent 并行工作，提高效率
  • 对抗式架构：Agent 相互竞争，提高鲁棒性
  • 联邦式架构：分布式 Agent 协作，保护隐私

Agent 通信机制

  • Agent 间通信：消息传递、共享内存、事件总线
  • Agent 与环境通信：API 调用、传感器数据、用户输入
  • Agent 与人通信：自然语言对话、可视化界面、混合交互

架构设计建议

  1. 根据任务特点选择合适的 Agent 架构
  2. 设计清晰的模块，每个模块职责单一
  3. 实现完善的通信机制
  4. 建立监控和日志系统
